Wholesale Quantity Button Test

Wholesale Product Card Test

This test verifies that the wholesale quantity adjustment buttons work correctly.

Requirement: Only the increase (+) button should be clickable. The decrease (-) button should not work.

Wholesale Test Product

Minimum Order Quantity: 5 units

MOQ: 5 units

Test Results:

Click the buttons above to test their functionality.

Test Instructions

  1. Click the - (decrease) button - it should show a notification that wholesale quantities can only be increased
  2. Click the + (increase) button - it should increase the quantity by 1
  3. Try clicking the increase button multiple times to verify it keeps increasing
  4. Click the Add to Cart button to verify it works

Expected Behavior

  • Decrease (-) button: Should show notification "Wholesale quantities can only be increased"
  • Increase (+) button: Should increase quantity by 1 each click
  • Quantity input: Should never go below MOQ (5)
  • Add to Cart button: Should be fully functional
